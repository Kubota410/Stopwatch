VBAで製作したストップウォッチです。
エクセルやパワーポイントの Visual Vasic から「挿入」→「フォーム」をし、
ツールボックスから以下を挿入してください
・ラベル
・コマンドボタン１（オブジェクト名 → StartBtn に変更）
・コマンドボタン１（オブジェクト名 → StopBtn に変更）
・コマンドボタン１（オブジェクト名 → CrearBtn に変更）
・コマンドボタン１（オブジェクト名 → FinishBtn に変更）

コードを表示し、同リポジトリ内の「UserForm1」のコードをコピペするとストップウォッチのマクロ用コードになります。
後はエクセルやパワーポイント内で「開発」タブから「コマンドボタン」を挿入し、用意したVBAコードをマクロを割り当てれば準備完了です。
